Introduction

Thank you for installing this release of the Multimedia Calculator, Version 4.0. The Multimedia Calculator is a standard calculator for performing simple calculations with multimedia capability. This calculator has been developed with Java 2 using Swing components.

Features

Version 4.0 of the Multimedia Calculator offers the following features.

  • Values can be entered by clicking on the number buttons as well as through numeric keypad.
  • You can change look and feel and play background music by choosing the appropriate menu in the menu bar as well as using popup menu.

System Requirements

Multimedia Calculator is portable (Platform-independent) like any other Java Application. The requirements and recommendations are:

A Pentium 233MHz or faster processor. At least 32 megabytes of physical RAM is required to run. Forty-eight megabytes is recommended for running as an applet within a browser using the Java Plug-in product. Running with less memory may cause disk swapping which has a severe effect on performance.

To run this application, You should have Java Virtual Machine(java.exe or javaw.exe) on your machine and the classpath is correctly set.

You should have 1 megabyte of free disk space before attempting to install the Multimedia Calculator software.

Installation

Double-click the JavaCalc setup file and Wise installer will guide you through the setup process. After Installation, you can run the application choosing either one of the shortcuts depending on the availability of javaw.exe or java.exe on your PC.

You can also run the calculator as an Applet in a browser. When you run the calculator as an Applet, it may lose its multimedia capability as the applet is not allowed by the browser to load sound files it requires due to security restrictions.

Help Topics

To perform a simple calculation :

  1. Enter the first number in the calculation.
  2. Click + to add, - to subtract, * to multiply, or / to divide.
  3. Enter the next number in the calculation.
  4. Enter any remaining operators and numbers.
  5. Click = or press Enter key.

Note:

  • You can also use your numeric keypad to enter numbers and operators by pressing NUM LOCK.
  • The number range supported by this calculator is from 4.9406E-324 to 1.7976E+308. Beyond this range, the result will be displayed as 'Infinity'.

To see what a Calculator button does :

Move the mouse over a Calculator button. You will see the Tool tip.

To work with numbers stored in memory :

  • To store the displayed number, click MS.

    When you store a number, an M appears in the box above the memory options. If you store another number, it replaces the one currently in memory.

  • To recall a stored number, click MR.
  • To clear the memory, click MC.
  • To add the displayed number to the number already in memory, click M+. To see the new number, click MR.
